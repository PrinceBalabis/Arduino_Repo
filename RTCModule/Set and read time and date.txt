How to set time and date
1. Open up Library Example: Serial_Easy
2. Set date and time variables in code 
3. Connect VCC to +5V
4. Upload code
5. Comment out
//	rtc.setDOW
//  rtc.setTime
//  rtc.setDate
6. Upload code again
5. Disconnect VCC/+5V cable

How to read time and date
1. Disconnect VCC/+5V cable
2. Open Serial Monitor
